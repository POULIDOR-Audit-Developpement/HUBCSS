# Support

## Usage Questions

Use GitHub Discussions if enabled. Otherwise open a GitHub issue for questions
that are clearly about package usage, documentation gaps, or integration
problems.

## Bug Reports

Open a bug report with:

1. The package version or commit.
2. The runtime or build environment.
3. A minimal reproduction.
4. The commands you ran and the observed result.

## Security Issues

For security concerns, follow `SECURITY.md` and avoid public disclosure first.
